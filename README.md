## 📌Overview

This is a portfolio project built with HTML, CSS, and JavaScript. The application features a live countdown to the 2026 FIFA World Cup and allows users to explore some of the tournament's top players.

Player information and trophy data are retrieved using the API-Football API⚽  

---

## 🔥Features
- Live World Cup countdown timer
- Player profile cards
- "Why Watch This Player" descriptions
- Career Journey section
- Trophy Cabinet section
- Responsive design
- Error handling for failed API requests

## API Endpoints
### Player Profiles
Retrieves:
- Player name
- Photo
- Nationality
- Age
- Position

### Trophies
Retrieves:
- Trophy name
- Season won

---

## ✅ How to Run
1. Clone this repository

 `git clone <repository-url>`

2. Open the project folder in VS Code.
3. Add your API-Football API key inside:

 `"x-apisports-key": "YOUR_API_KEY"`
 
4. Open `api-worldcup.html` in your browser.
